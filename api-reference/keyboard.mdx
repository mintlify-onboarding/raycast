The Keyboard APIs are useful to make your actions accessible via the keyboard shortcuts. Shortcuts help users to use your command without touching the mouse.

## Types

### Keyboard.Shortcut

A keyboard shortcut is defined by one or more modifier keys (command, control, etc.) and a single key equivalent (a character or special key).

See [KeyModifier](/api-reference/keyboard#keyboard.keymodifier) and [KeyEquivalent](/api-reference/keyboard#keyboard.keyequivalent) for supported values.
